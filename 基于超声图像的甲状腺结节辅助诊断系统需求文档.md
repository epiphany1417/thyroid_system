# 基于超声图像的甲状腺结节辅助诊断系统需求文档

## 一、项目概述

### 1.1 项目背景

甲状腺结节是临床常见疾病，其中部分结节具有恶性风险（甲状腺癌）。超声检查是主要诊断手段，但其判读依赖医生经验，存在主观性强、效率低的问题。随着人工智能技术的发展，基于深度学习的医学影像辅助诊断系统能够帮助医生进行结节检测与良恶性判断，提高诊断效率与准确性。

### 1.2 项目目标

设计并实现一个完整的甲状腺结节辅助诊断系统，系统需包含用户管理、数据管理、AI分析等模块，体现真实医疗系统的基本结构。

## 二、系统总体要求

- 模拟一个简化的医院辅助诊断平台
- 支持多用户角色使用
- 具备基本的医学影像分析能力
- 基于公开医学影像数据集（如 TN5000）

## 三、用户角色设计

系统必须包含三类用户：

### 3.1 医生模块（核心用户）

**功能需求：**

1. 登录系统
2. 上传患者超声图像
3. 查看AI诊断结果：
   - 结节位置（检测框）
   - 良恶性分类（benign/malignant）
   - 风险概率（0~1）
4. 手动输入或修改诊断意见（可选）
5. 查看历史诊断记录
6. 按患者查询记录

**现实对应：** 放射科医生 / 超声科医生

### 3.2 患者模块

**功能需求：**

1. 注册与登录
2. 上传个人检查图像（可选）
3. 查看自己的诊断结果：
   - 图像、AI结果、医生意见（如有）
   - 查看历史记录
   - 简单结果解释（如"风险较低/较高"）

**现实对应：** 普通就诊患者

### 3.3 管理员模块

**功能需求：**

1. 用户管理：创建/删除用户
2. 数据管理：查看系统所有影像数据、删除异常数据
3. 系统监控：日志查看、操作记录
4. 模型管理（简化）：更新模型（可选）、查看模型版本信息

**现实对应：** 医院信息系统管理员

## 四、系统功能模块

### 4.1 用户管理模块

- 登录 / 注册
- 角色权限控制
- 用户信息管理

### 4.2 数据管理模块

- 医学图像上传（jpg/png）
- 图像存储（本地或数据库）
- 标注数据管理（XML解析）
- 数据查询（按用户/时间/类型）

### 4.3 图像处理模块

- 图像显示
- 标注框（bbox）可视化
- 图像缩放与浏览
- 可选：ROI手动选择

### 4.4 AI辅助诊断模块（核心）

- **输入：** 超声图像
- **处理：** 使用已训练模型
- **输出：**
  - 检测框（结节位置）
  - 分类结果：benign（良性）/malignant（恶性）
  - 概率值（0~1）

### 4.5 结果展示模块

- 图像 + 检测框显示
- 分类结果展示
- 风险概率可视化（进度条/柱状图）
- 简单文字解释：
  - 检测结果：malignant
  - 风险评估：高风险，建议进一步检查

### 4.6 日志与记录模块

- 用户操作日志
- 诊断历史记录
- 支持查询：按用户、按时间、按诊断结果
